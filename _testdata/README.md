# 图片上传功能测试用文章

## 测试场景覆盖

| 文件 | 场景 | 章节数 | 测试重点 |
|------|------|--------|----------|
| 1-short-travel-2sections.txt | 短文，旅行类 | 2 | 少量章节，图片正好覆盖所有节 |
| 2-long-tech-5sections.txt | 长文，科技类 | 5 | 多章节，部分选部分不选 |
| 3-mid-tutorial-3sections.txt | 中等，教程类 | 3 | 3 节+不选章节的红字提示 |
| 4-single-section-edgecase.txt | 极短，单节 | 1 | 只有第1节可选，选第2节应静默跳过 |

## 测试方法

1. `npm run dev` 启动后打开页面
2. 粘贴文章内容，点「开始智能排版」等 AI 分析完成
3. 上传 2-3 张测试图片（可用手机随手拍的照片）
4. 给不同图片选不同章节号（覆盖：选了第1节、选了第2节、不选章节）
5. 点「开始智能排版」等出结果
6. 检查：选了章节的图是否出现在对应章节末尾，不选的图是否没出现
7. 分别测试「复制 HTML」「导出 HTML」「导出长图」，确认图片包含在内
